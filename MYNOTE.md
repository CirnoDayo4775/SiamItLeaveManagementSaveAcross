# Note ถึงตัวเองเวลาอ่าน

## ขั้นตอนการเปิดใช้งาน

### ก่อนเริ่มระบบ
1. ตรวจสอบ [Backend/.env](\siamit\Backend\.env) ว่า DB_PASSWORD= ว่างไหม ต้องทำให้ว่าง

### เริ่มระบบ
1. ใช้ cd Backend -> node index.js เพื่อเริ่มระบบหลังบ้าน
2. ใช้ cd frontend -> npm run dev เพื่อเริ่มระบบหน้าบ้าน

อย่าลืมอัพเดทไฟล์นี้เพิ่มเติมทุกครั้งที่มีอะไรอัพเดท
